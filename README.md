## 📌 Descripción

Gourmet Go es una aplicación web que permite buscar recetas de cocina de forma dinámica utilizando una API externa.  
El usuario puede ingresar ingredientes en español y obtener recetas relacionadas con imágenes, instrucciones y detalles relevantes.

Este proyecto fue desarrollado como parte de mi formación como **Desarrolladora Full Stack JavaScript Trainee**, 
enfocándome en el consumo de APIs, manipulación del DOM y experiencia de usuario.

---

## 🚀 Funcionalidades

- 🔎 Búsqueda de recetas por ingrediente
- 🌎 Traducción de ingredientes (ES → EN) para mejorar resultados
- 🖼️ Visualización de recetas con imágenes
- 📋 Detalle de preparación
- ⚡ Renderizado dinámico sin recargar la página

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (Vanilla JS)
- API: TheMealDB
